HTML, CSS, JS Creative Project. Small project to experiment and learn HTML, CSS and JS.

Instructions:
To play the game, download the zip and open up "index.html"
Game is meant to be played on a window size of 1280x800.
A image file is included for reference. Open the image and resize the window using the photo as a reference.
